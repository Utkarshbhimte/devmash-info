FAQ

  • What is Github Account and why is it necessary?Github is Git Repository Hosting Service. Developers put their code online to help other or to further improve the code. Every Developer should be familiarized with it, that is why we encourage our students to have a Github Account. You can create an account there for free and it takes only a couple of minutes.
  • When will the classes start?August 11 will be our Inauguration Date, The classes will start from 20th of August.
  • How long will these classes be?The format of This Course is different from the traditional way. The Classes will be assisting. We won't be teaching you from the scratch. You should be having basic knowledge of Syntax and Object Oriented Programming. We will have Weekly sessions to talk on topics which require special attention, Helping with clearing out doubts and Project Review. We will provide content which will help you learn the whole course.
